# DBOGLOBAL
 DBOG Client & Server Emulator

## 3rd Party
- Boost 1.64.0 - Required for Server (Will be removed soon)
- MySQL 5.7.18 - Required for Server
- Xtreme ToolkitPro v15.2.1 - Required for Tools
- GFx SDK 3.3 - Required for Client

## Requirements
- DirectX9 SDK
- Visual Studio 2019
